## Description

ALLIANZ READ FILE LON SCHEDULE

USE NODE VERSION 20.12.0 [NodeJs](https://nodejs.org/en/blog/release/v20.12.0)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run build
$ npm run start:prod
```

## Setup step

1. Install Nodejs
2. Create env from env.example
    - LOCATION_PATH is path of all template
    - PAGE_ID and LON_API_TOKEN in example is for test page contract zwiz for production config
3. Install package run command "npm install"
4. Build project run command "npm run build"
5. Start project run command "npm run start:prod"
